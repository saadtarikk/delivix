# Delivix New

A WordPress-based project for Delivix.

## Project Structure

This project contains a WordPress installation with the following structure:

- `app/public/` - WordPress root directory
- `conf/` - Configuration files for nginx, MySQL, and PHP
- `logs/` - Log files for various services

## Setup

### Prerequisites

- Local development environment (e.g., Local by Flywheel, XAMPP, or similar)
- PHP 7.4 or higher
- MySQL 5.7 or higher
- Nginx or Apache web server

### Installation

1. Clone this repository
2. Configure your local development environment
3. Import the database if needed
4. Update WordPress configuration files
5. Access the site through your local development URL

## WordPress

This is a standard WordPress installation. Key directories:

- `wp-content/themes/` - WordPress themes
- `wp-content/plugins/` - WordPress plugins
- `wp-content/uploads/` - Media uploads
